5) Betrachten Sie die Meldungen von PMD.
Wie unterscheiden sich diese von Checkstyle?
Checkstyle: Schönheitssachen, betrachtet Code als Test
 --> syntaktisch
PMD: Logiksachen, prüft Code auf Sinn, Zusammenhang mit anderem Code
 -->semantisch
